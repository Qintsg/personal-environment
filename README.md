# Personal Environment

这个仓库用于记录和同步个人电脑环境，包括已安装软件、独立配置、初始化步骤和常用维护说明。目标是在更换设备、重装系统或回溯配置时，能快速恢复一套可用的工作环境。

## 内容范围

- 软件清单：记录常用应用、开发工具、驱动和系统组件。
- 配置文件：保存可复用、可公开的配置片段，不提交密钥和本地隐私数据。
- 安装步骤：记录系统初始化、软件安装和环境变量配置流程。
- 维护文档：说明仓库目录规则、命名方式和更新约定。

## 推荐目录

- `docs/`：仓库规范、安装说明和维护文档。
- `apps/`：软件记录，每个软件一个独立目录。
- `configs/`：独立配置文件，按程序名分目录保存。
- `scripts/`：可复用脚本，按系统或用途分组。
- `notes/`：零散经验、排障记录和补充说明。

当前目录可以按需要逐步补齐，不需要为了占位创建空目录。

## 维护原则

- 每个程序的配置尽量放在独立目录中，避免混杂。
- 文档优先写清楚用途、来源路径和恢复方法。
- 不提交 `.env`、token、私钥、账号密码、授权文件等敏感内容。
- 删除不用的旧目录，保持结构简单。

## 同步脚本

- 使用 `scripts/export-packages.ps1` 同步本机软件包清单、包管理器原始导出和可恢复安装记录。
- 使用 `scripts/export-configs.ps1` 同步白名单内的常用配置文件，并在写入前做通用脱敏。
- 导出后优先复核 `apps/package-export-summary.md` 和 `apps/config-export-summary.md`。
