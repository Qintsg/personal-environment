# 百度网盘

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：8.3.9
- 包 ID：`Baidu.BaiduNetdisk`

## 安装方式

```powershell
winget install --id Baidu.BaiduNetdisk --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 不在仓库中记录账号或同步目录内容。
