# GitHub CLI

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：2.91.0
- 包 ID：`GitHub.cli`

## 安装方式

```powershell
winget install --id GitHub.cli --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

- 本机已检测到 `gh` 相关安装记录。
