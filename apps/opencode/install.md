# OpenCode

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：1.14.29
- 包 ID：`SST.OpenCodeDesktop`

## 安装方式

```powershell
winget install --id SST.OpenCodeDesktop --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```
