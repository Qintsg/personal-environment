# UU远程

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：4.21.0.7755
- 包 ID：`NetEase.UURemote`

## 安装方式

```powershell
winget install --id NetEase.UURemote --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注
