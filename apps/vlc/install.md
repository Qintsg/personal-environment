# VLC media player

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：3.0.23
- 包 ID：`VideoLAN.VLC`

## 安装方式

```powershell
winget install --id VideoLAN.VLC --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

- 当前记录来自 Windows 已安装程序清单。
