# Termius

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：9.37.6
- 包 ID：`Termius.Termius`

## 安装方式

```powershell
winget install --id Termius.Termius --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 不在仓库中记录 SSH 主机、私钥、账号或同步数据。
