# NVIDIA CUDA Toolkit

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：13.2
- 包 ID：`Nvidia.CUDA`

## 安装方式

```powershell
winget install --id Nvidia.CUDA --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 本机存在多个 CUDA Toolkit 版本记录。
