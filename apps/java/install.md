# Java

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：25.0.2.0
- 包 ID：`Oracle.JDK.21`

## 安装方式

```powershell
winget install --id Oracle.JDK.21 --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 本机存在多个 Java 版本；旧版本是否需要保留需按项目确认。
