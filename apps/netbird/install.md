# Netbird

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：0.67.1
- 包 ID：`Netbird.Netbird`

## 安装方式

```powershell
winget install --id Netbird.Netbird --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 不在仓库中记录网络凭据或设备密钥。
