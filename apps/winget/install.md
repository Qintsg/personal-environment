# Windows Package Manager

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：1.29.179.0
- 包 ID：`Microsoft.AppInstaller`

## 安装方式

```powershell
winget install --id Microsoft.AppInstaller --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- `winget` 通常随 App Installer 提供。
