# 1Password

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：> 8.12.12.44
- 包 ID：`AgileBits.1Password`

## 安装方式

```powershell
winget install --id AgileBits.1Password --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 不在仓库中记录任何密码库或密钥内容。
