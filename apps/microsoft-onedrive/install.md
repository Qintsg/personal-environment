# Microsoft OneDrive

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：26.070.0414.0001
- 包 ID：`Microsoft.OneDrive`

## 安装方式

```powershell
winget install --id Microsoft.OneDrive --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```
