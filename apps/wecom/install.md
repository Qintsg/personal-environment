# 企业微信

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：5.0.8.6006
- 包 ID：`Tencent.WeCom`

## 安装方式

```powershell
winget install --id Tencent.WeCom --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 不在仓库中记录账号或工作区数据。
