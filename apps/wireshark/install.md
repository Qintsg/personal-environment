# Wireshark

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：4.6.5
- 包 ID：`WiresharkFoundation.Wireshark`

## 安装方式

```powershell
winget install --id WiresharkFoundation.Wireshark --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 本机同时存在 Npcap 和 USBPcap 记录。
