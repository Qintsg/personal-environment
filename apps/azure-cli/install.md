# Microsoft Azure CLI

## 默认平台

默认安装平台为 Windows；本文件中的安装命令默认在 Windows PowerShell 中执行。

## 当前记录

- 来源：winget
- 当前版本：2.85.0
- 包 ID：`Microsoft.AzureCLI`

## 安装方式

```powershell
winget install --id Microsoft.AzureCLI --exact --accept-package-agreements --accept-source-agreements --disable-interactivity
```

## 备注

- 不在仓库中记录 Azure 登录凭据、订阅信息或 token。
